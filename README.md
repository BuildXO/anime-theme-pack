You can further customize any theme by adding overrides to your `settings.json`:

```json
{
  "workbench.colorCustomizations": {
    "[Jujutsu Kaisen: Gojo Satoru]": {

    "[Jujutsu Kaisen: Gojo Satoru]": {
      "comments": "#6E7681"
    }
  }
}
```
