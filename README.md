# 🎌 Anime Theme Pack

A comprehensive collection of VS Code themes inspired by popular anime series. Each theme features carefully crafted color palettes with extensive UI customization, similar to the Doki Theme suite.

## 🎨 Available Themes

### Jujutsu Kaisen
- **Gojo Satoru** - Dark theme featuring Gojo's signature limitless blue color scheme with 200+ customized colors

### Attack on Titan
- **Eren's Determination** - Dark theme with earthy brown tones representing the fight for freedom

### My Hero Academia
- **Deku Plus Ultra** - Dark theme with vibrant green colors inspired by One For All

### Naruto
- **Hokage Orange** - Dark theme with bold orange representing the Will of Fire

### One Piece
- **Straw Hat Red** - Dark theme with adventurous red tones for the pirate king

### Demon Slayer
- **Sakura Breathing** - Dark theme with soft pink and purple hues

### Cyberpunk
- **Edgerunners: Neon Night** - Dark theme with neon cyberpunk aesthetics
- **Akira: Neo Tokyo** - Dark theme inspired by Neo Tokyo

### Classic
- **Animee Classic** - Original Material-inspired dark theme

## ✨ Features

- **200+ Color Definitions** per theme for comprehensive UI customization
- **Extensive Token Colors** for rich syntax highlighting across all languages
- **Semantic Highlighting** support for modern language features
- **Carefully Crafted Palettes** inspired by each anime's signature colors
- **Consistent Experience** across all VS Code UI elements

## 🎯 Theme Structure

Each theme includes:
- Complete editor color customization
- Activity bar, sidebar, and status bar theming
- Terminal color schemes with ANSI support
- Git decoration colors
- Debug toolbar styling
- Notification and panel theming
- Comprehensive symbol icon colors
- Breadcrumb and peek view customization

## 🚀 Installation

1. Open VS Code
2. Go to Extensions (Ctrl+Shift+X / Cmd+Shift+X)
3. Search for "Anime Theme Pack"
4. Click Install
5. Select your preferred theme from Command Palette (Ctrl+K Ctrl+T / Cmd+K Cmd+T)
