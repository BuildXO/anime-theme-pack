# 📸 Screenshots Folder

This folder should contain preview images for each theme.

## Current Status

The `.placeholder` files are temporary - they indicate where actual theme screenshots should be placed.

## How to Add Screenshots

1. **Activate a theme** in VS Code
2. **Open a sample code file** (JavaScript, Python, etc.) to showcase syntax highlighting
3. **Take a screenshot** (recommended size: 1920x1080 or 1280x720)
4. **Save as PNG** with the appropriate filename:
   - `jjk-gojo.png`
   - `attack-on-titan.png`
   - `my-hero-academia.png`
   - `naruto.png`
   - `one-piece.png`
   - `demon-slayer.png`
   - `cyberpunk-neon.png`
   - `akira.png`
   - `animee-classic.png`

5. **Delete the corresponding `.placeholder` file**

## Tips for Good Screenshots

- Use a code file that shows various syntax elements (keywords, strings, functions, comments)
- Make sure the editor is maximized for clarity
- Include the sidebar and activity bar to show the full theme
- Avoid personal information in the code
- Use a clean workspace without notifications or popups
